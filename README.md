# Orça Agora

Projeto confidencial Orça Agora

## Backend

Slimframework, MySQL

## Frontend

React, Bootstrap

## GET /api/{param}

Retorna conteúdo da tabela {param}
array

## POST /api/{param}

Insere conteúdo na tabela
Retorna conteúdo da última linha modificada
Renderização de página

## PUT /api/{param}/{id}

Slim error, variável indefinida linha 103

## DELETE /api/{param}/{id}

String

## GET /validation/mail/{email}

Rota vazia

## GET /validation/document/{doc}

Retorna as informações de documento
Bool

## GET /validation/mail/{userName}/generate

Gera o hash do usuario
Array

## GET /validation/mail/{userName}/verify

Verifica as informações do hash
Bool

Renderização de página
